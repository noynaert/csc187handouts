# 05.040 The Math Class

Looking at the API for the math class

## Constants

* E   (Natural Log)
* PI

```java
System.out.println(Math.PI);
System.out.println(Math.E);
```

## Math methods to know

* Math.abs()
* Math.sqrt()
* Math.max()
* Math.min()
* Math.random()  Returns a value r such that
  
  $0.0 \le r \lt 1.0$
